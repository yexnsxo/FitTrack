득근득근
나만의 운동 루틴을 계획하고, 타이머와 함께 수행하며, 캘린더로 꾸준함을 기록하는 안드로이드 운동 보조 앱입니다.
<br>
✨ 주요 기능
•
✅ 오늘의 운동 관리 (To-do)
◦
오늘 수행할 운동 목록을 추가하고 관리할 수 있습니다.
◦
운동 종류(무게, 횟수 등)와 목표치를 설정하여 체계적인 계획을 세울 수 있습니다.
•
⏱️ 백그라운드 운동 타이머
◦
운동 목록에서 타이머를 시작하여 세트, 휴식 시간을 관리할 수 있습니다.
◦
Foreground Service를 활용하여 앱이 백그라운드에 있거나 화면이 꺼져도 타이머가 중단되지 않고 알림을 통해 현재 상태를 확인할 수 있습니다.
•
🗓️ 운동 기록 및 캘린더
◦
완료된 운동은 날짜별로 자동 기록됩니다.
◦
캘린더 뷰를 통해 언제 운동을 했는지 한눈에 파악하고 성취도를 확인할 수 있습니다.
<br>
🛠️ 기술 스택
•
UI: Jetpack Compose, Material 3
•
Architecture: MVVM (Model-View-ViewModel)
•
Asynchronous: Kotlin Coroutines
•
Navigation: Navigation Compose
•
Local DB: Room
•
Background Processing: Foreground Service
•
DI: ViewModelFactory를 사용한 수동 의존성 주입
•
Libraries:
◦
Coil: 이미지 로딩
◦
Kizitonwose Calendar: 캘린더 UI 구현
◦
Kotlinx Serialization: 데이터 직렬화
